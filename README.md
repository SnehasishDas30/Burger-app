# 🍔 Tasty Burger - Food Ordering Web App

## 📌 Overview

**Tasty Burger** is a modern and responsive food ordering web application built with **React.js**. It allows users to browse delicious burgers, add products to the cart, manage a wishlist, place orders, and view order history through a clean and user-friendly interface.

The application uses **LocalStorage** for authentication and data persistence, making it fully functional without a backend server.

---

## 🚀 Features

* 🔐 User Login & Signup
* 👤 User Profile
* 🍔 Browse Burger Menu
* 🛒 Add to Cart
* ❤️ Wishlist Management
* 📦 Checkout with Form Validation
* 📋 My Orders Page
* 🔔 Toast Notifications
* 💾 LocalStorage Data Storage
* 🚪 Login & Logout Functionality
* 📱 Fully Responsive Design

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/SnehasishDas30/tasty-burger.git
```

Go to the project folder:

```bash
cd tasty-burger
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Open your browser:

```text
http://localhost:3000
```
